# 🧠 Quiz App

A simple **Flutter Quiz Application** that allows users to answer multiple-choice questions and view their final score.

---

## ✨ Features
- Multiple choice quiz questions
- Clean and simple UI
- Instant answer selection
- Final score screen
- Restart quiz

---

## 🛠 Built With
- Flutter
- Dart
